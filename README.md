**To-Do List — Plan Smart. Do Better.**  

A simple and responsive To-Do List Web App built using Java (backend) and HTML, CSS, JavaScript (frontend).  
It allows users to add, delete, and mark tasks as complete — all managed by a lightweight local Java HTTP server.

## Features

- Add new tasks instantly  
- Delete completed or unwanted tasks  
- Toggle task completion (done or active)  
- Clear all completed tasks with one click  
- Responsive, modern glass-style UI  
- Lightweight Java backend with no database required  

---

## Technologies Used

- Java (Core) – Backend logic and HTTP server  
- HTML5 – Page structure and content  
- CSS3 – Styling and UI design  
- JavaScript (Fetch API) – Dynamic frontend interactions  
- Git & GitHub – Version control and hosting

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/vetrikarunanithi/to-do-list.git
   ```
2. Navigate to the project directory:
   ```bash
   cd to-do-list
   ```
3. Open the project in VS Code or any IDE.
4. Run the Java backend:
   ```bash
   cd src
   javac Main.java
   java Main

   To-Do List Server Started
   Visit: http://localhost:8080
   ```
5. Run the frontend:
- Open frontend/index.html in your browser
- Or use the Live Server extension in VS Code

---

## Usage
- Enter a new task and click Add
- Check or uncheck tasks to mark them complete or active
- Use the delete icon to remove a specific task
- Click Clear Completed to remove all completed tasks
- Filter between All, Active, and Completed tasks
